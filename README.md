# Online Book Store SQL Project

## Project Overview
This project is based on an Online Book Store database system using PostgreSQL and pgAdmin 4.  
The goal of this project is to analyze customer orders, book sales, revenue, and inventory data using SQL queries.

This project demonstrates real-world SQL problem-solving and data analysis skills.

---

## Tools & Technologies Used
- PostgreSQL
- pgAdmin 4
- SQL
- CSV Dataset

---

The project contains the following tables:

## Business Problems Solved

### Basic Level Queries
- Retrieve all books in a specific genre
- Find books published after a certain year
- List customers from a specific country
- Show orders placed in a particular month

### Intermediate Level Queries
- Find total revenue generated
- Identify the most expensive book
- Calculate total books sold
- Find top customers by purchase amount
- Retrieve the most frequently ordered books

### Advanced Level Queries
- Analyze customer purchasing behavior
- Find highest selling genres
- Calculate remaining stock after orders
- Revenue analysis by category
- Order trend analysis

- Repeat customers contributed significantly to revenue.
- Some books had low stock after frequent orders.
- Customer ordering patterns helped identify popular categories.
